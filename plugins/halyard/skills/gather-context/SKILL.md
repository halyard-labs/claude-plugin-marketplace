# Gather Context

Before starting work, gather context from available sources:

1. **Knowledge base** — Use `search_knowledge` and `explore_knowledge` for prior decisions, past work, and expert Q&A
2. **Codebase** — Explore relevant files, git history, and project documentation
3. **Web** — Search for external context when the topic extends beyond the repo

If knowledge base tools are unavailable, rely on codebase exploration and web search. Synthesize findings into a brief summary before proceeding.
